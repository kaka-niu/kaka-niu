🚀 部署步骤

1

创建同名仓库

在 GitHub 上创建一个仓库，仓库名必须和你的用户名完全一致：kaka-niu/kaka-niu，设为 Public，勾选「Initialize with README」。

2

上传 README.md

把 README.md 内容替换进这个仓库。保存后，主页顶部立刻显示内容。

3

生成 Personal Access Token

GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → New token。
权限勾选：user → read and write。复制 token 备用。

4

添加 Repository Secret

在 kaka-niu/kaka-niu 仓库 → Settings → Secrets → Actions → New secret。
名称：GH_TOKEN，值：粘贴上一步的 token。

5

上传脚本和 Action 文件

把 update_bio.py 放到仓库根目录，把 .github/workflows/update-bio.yml 放到对应目录。

6

验证运行

进入 Actions 选项卡，找到「Auto Update Bio」，点击「Run workflow」手动触发一次，确认成功后，每小时自动运行。
